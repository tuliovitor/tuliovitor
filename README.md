É importante destacar que esse é um arquivo .dm, que indica que é um arquivo markdown e todo esse texto mostrado acima está em markdown (que positivamente se parece muito com um texto que digitamos no whatsapp ou no word). Segundo o tecnoblog:

Markdown é uma linguagem voltada para formatação de textos. Com ela, você não precisa apertar um botão na interface para deixar o texto em itálico ou sublinhado, por exemplo. Em vez disso, é só colocar alguns marcadores enquanto digita, como underlines (_) e asteriscos (*). — Tecnoblog, em "O que é markdown?"

Para quem já usou HTML o markdown tem várias coisas bem parecidas e até algumas limitações em relação ao HTML (que vamos falar mais pra frente). A grande vantagem do markdown é exatamente sua simplicidade para tornar textos comuns bem mais bonitos e amigáveis.

Cabeçalhos / Títulos (heading)
Começando a falar do arquivo, ele vai começar com 3 asteriscos (###) seguido de um texto “Hi there 👋” (esse é exatamente o mesmo texto que estava aparecendo no seu perfil). Esse 3 asteriscos indicam que você está utilizando um heading (ou cabeçalho / título na tradução literal) "nível" 3

Esses cabeçalhos vão ser utilizados para titúlos, subtítulos, etc e é o equivalente no HTML as tags H1, H2, H3, etc. O que vai diferenciar a quantidade de asteriscos no markdown é o tipo de título que você deseja utilizar. Para títulos podemos utilizar o #, para subtítulos o ## e assim por diante.

Visualmente quando maior o número de asteriscos menor vai ser o texto, além disso o # e o ## possuem uma linha fina embaixo deles, como mostrado na figura:

Press enter or click to view image in full size

Comparação entre os títulos (heading) no markdown utilizando #, ##, ###,…
Em sequência, logo após esse título “Hi there 👋” aparece um "<!--", uma sequência de textos e no final um "-->". Esses dois elementos representam um comentário no markdown (o "<!--" abre o comentário e o "-->" fecha) e tudo que estiver entre eles não vai ser exibido para o usuário.

Nesse caso, todo esse texto mostrado serve apenas para sugerir ao usuário o que ele pode colocar no seu perfil.

Você pode simplesmente apagar tudo isso e começar a colocar o que faz sentido pra você.

Links
Além disso, o markdown também permite que você utilize links para que o usuário clique e seja direcionado para suas redes sociais, algum artigo que você ache interessante ou até pro link do seu currículo.

Para criar um link no markdown você pode passar entre colchetes "[]" o texto para ser exibido e então entre parênteses “()” qual vai ser o link que o usuário será direcionado

Um link para direcionar o usuário para o Google poderia ser escrito como " "[Clique para acessar o Google](www.google.com.br)". Na próxima imagem é possível visualizar o link e como ele vai aparecer na sua página.

Estilos do texto
Outra ferramenta muito legal que o markdown oferece é uma maneira fácil de estilizar o texto e deixá-lo mais atrativo, colocando elementos em negrito, alguns em itálico e outros tanto em negrito quanto em itálico.

Para colocar um texto em negrito, basta você colocar o texto entre 2 asteriscos, da seguinte forma: **esse é um texto em negrito** e o resultado do markdown será exatamente esse é um texto em negrito. Já para colocar um texto em itálico você deve usar apenas 1 asterisco *esse é um texto em itálico*, resultando em esse é um texto em itálico.

Ao utilizar 3 asteriscos, você teria o texto tanto em negrito quanto em itálico: ***esse é um texto em negrito e itálico*** ficaria esse é um texto em negrito e itálico.

Depois de fazer qualquer edição, você pode clicar em em "Preview" e visualizar o resultado que está sendo gerado, já com os textos em negrito, os links e os headers.
